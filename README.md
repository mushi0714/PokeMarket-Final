# 🏛️ PokéMarket - Trainer Management System

Professional management system for Pokémon Trainers developed as part of a Final Evaluation. This application integrates real-time data from the PokéAPI with a custom local backend to provide a seamless market experience.

## 🚀 Key Features

* **Hybrid Data Source:** Combines official data from PokéAPI with exclusive local products via `json-server`.
* **Role-Based Access Control (RBAC):** Secure routes for `Clients` and `Admins`.
* **Trainer Profile:** Personalized cards with gender-based avatars and persistent wishlists.
* **Smart Shopping Cart:** Vertical management system with real-time total calculation.
* **Glassmorphism UI:** Modern, clean interface using a matte professional color palette.

## 🛠️ Tech Stack

* **Frontend:** React.js, React Router, Lucide React (Icons).
* **State Management:** Context API (AuthContext).
* **Styling:** CSS3 (Custom Glassmorphism properties).
* **Notifications:** Sonner.
* **Backend Simulation:** JSON Server.
